🎯 JobAgent 智能求职助手
基于 NLP 与多智能体协作的一站式求职系统
✨ 项目介绍
JobAgent 是一款基于 LangGraph 多智能体架构 + 大语言模型 构建的智能求职辅助系统，能够自动完成简历分析、求职信生成、职业路径规划、公司背景查询、行业趋势分析、面试攻略等全流程求职服务，帮助求职者高效、精准、智能化完成求职全过程。
本项目包含 本地多智能体系统（MultiAgent） + 云端轻量化助手（Coze） 双架构，兼顾深度定制与便捷使用，满足不同用户需求。
🚀 核心功能
✅ 智能简历解析自动读取 PDF 简历，提取教育背景、实习经历、项目经验、技能栈，生成结构化总结。
✅ 专属求职信生成基于简历内容自动生成高度定制化求职信，无需手动输入。
✅ 职业路径可视化规划根据个人背景自动生成 0-1 年、1-3 年、3-5 年职业发展路线图。
✅ 公司 / 行业 / 面试攻略无需联网，基于大模型知识库提供公司背景、行业趋势、面试高频题与回答技巧。
✅ 多智能体协同Supervisor 智能调度 + 简历分析器 + 求职信生成器，实现任务自动接续。
🛠 技术栈
前端界面：Streamlit
多智能体框架：LangGraph
大模型支持：通义千问 / Qwen
文档解析：PyPDF2
开发语言：Python 3.10+
工具链：LangChain、Prompt 工程、状态管理、任务路由

🔧 快速启动
1. 安装依赖
bash
运行

2. 配置大模型 API
在 .env 文件中填写：
plaintext
DASHSCOPE_API_KEY=你的API密钥
3. 启动系统
bash
运行
streamlit run app.py
